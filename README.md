# GenPlatform.io

Official landing page for **Generation Tech Enterprise** — the parent 
infrastructure company for GenGuard, GenSchool, and GenNote.

## Tech Stack
- Next.js 14 (App Router)
- Tailwind CSS v3
- Lucide React Icons
- TypeScript

## Products
| Product | Description | Status |
|---------|-------------|--------|
| GenGuard | AI WhatsApp fraud detection bot | Public Beta |
| GenSchool | School management platform | Early Access |
| GenNote | AI voice notes & transcription | Private Beta |

## Features
- Dark mode industrial design (bento grid layout)
- CSS-only product mockups (no external images)
- Scroll-triggered reveal animations
- Mouse-following glow card effects
- Animated dot grid hero background
- Scroll progress bar + back-to-top
- Slide-out mobile navigation
- Production-ready Privacy Policy & Terms of Service
- Compliance with Ghana Data Protection Act, 2012 (Act 843)
